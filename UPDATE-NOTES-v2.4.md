# Was ist neu in Version 2.4?

Hallo! Hier ist eine kurze Übersicht, was sich mit dem Update auf Version 2.4 geändert hat.
Das Update wird automatisch im Hintergrund eingespielt — du musst nichts manuell tun.

---

## Das Wichtigste auf einen Blick

### Verlorene Tode werden automatisch nachgetragen

Bisher konnte es passieren, dass ein Tod nicht gezählt wurde, wenn der Death Tracker gerade
nicht lief (z. B. weil der PC neu gestartet wurde, während du gespielt hast). Das ist jetzt
Geschichte: Beim nächsten Start prüft der Tracker selbst, ob Tode fehlen — und trägt diese
still nach. Deine Statistiken bleiben immer vollständig.

### Das LOTRO-Plugin aktualisiert sich jetzt mit

Bisher wurde beim automatischen Update nur der Tracker selbst aktualisiert. Das Plugin
innerhalb von Herr der Ringe Online musste gelegentlich manuell kopiert werden. Ab jetzt
passiert das automatisch mit — du musst nichts weiter tun.

### Updates werden nicht mehr mitten im Spiel eingespielt

Wenn du spielst und ein Update verfügbar ist, fragt der Tracker dich jetzt höflich, wann
ein guter Zeitpunkt wäre:
- **Jetzt installieren** — das Spiel wird kurz geschlossen und der Tracker aktualisiert
- **In 3 Stunden erinnern** — der Tracker fragt später nochmal nach
- **Beim nächsten LOTRO-Start erinnern** — er wartet, bis du das Spiel das nächste Mal öffnest

So entscheidest du selbst, wann das Update passiert.

### Das Overlay zeigt keine Tode mehr doppelt an

In seltenen Fällen konnte es vorkommen, dass ein Tod zweimal im Overlay angezeigt wurde.
Dieser Fehler ist behoben.

---

## Kleinere Verbesserungen

- Der Tracker startet nach einem unerwarteten Absturz jetzt zuverlässiger neu, ohne
  dass du eingreifen musst.
- Die Zeitangaben in den Protokolldateien zeigen jetzt deine lokale Uhrzeit statt
  der weltweiten Standardzeit (UTC) — das macht die Logs viel leichter lesbar.
- Updates werden jetzt sicherer eingespielt: Sollte während eines Updates der Strom
  ausfallen oder der PC abstürzen, stellt der Tracker beim nächsten Start die Dateien
  automatisch wieder her.

---

## Was du tun musst

**Nichts.** Das Update passiert vollautomatisch im Hintergrund, sobald du den PC das
nächste Mal startest. Du bekommst einen kurzen Hinweis, wenn alles fertig ist.

Das war's!
